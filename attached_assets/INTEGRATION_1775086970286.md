# Adding MetahersHeroVideo to Your React App

## Files in this folder

| File | What it is |
|---|---|
| `MetahersHeroVideo.tsx` | The complete animated component — copy this one file |
| `MetahersHeroVideo.css` | Optional brand CSS variables (only needed for styling other elements) |

---

## Step 1 — Install framer-motion

In your other Replit project's Shell, run:

```bash
npm install framer-motion
# or
yarn add framer-motion
# or
pnpm add framer-motion
```

---

## Step 2 — Add Google Fonts

Open your project's `index.html` (or equivalent HTML entry point) and add
this inside the `<head>` tag:

```html
<link
  rel="preconnect"
  href="https://fonts.googleapis.com"
/>
<link
  rel="preconnect"
  href="https://fonts.gstatic.com"
  crossorigin
/>
<link
  href="https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;500;600&family=Inter:wght@300;400;500&display=swap"
  rel="stylesheet"
/>
```

---

## Step 3 — Copy the component

Copy `MetahersHeroVideo.tsx` into your project's `src/components/` folder
(or wherever you keep shared components).

---

## Step 4 — Use it

```tsx
import MetahersHeroVideo from './components/MetahersHeroVideo';

// Full-viewport hero section:
export default function HeroSection() {
  return (
    <div style={{ width: '100%', height: '100vh' }}>
      <MetahersHeroVideo />
    </div>
  );
}
```

```tsx
// Inside a fixed-height card or panel:
<div style={{ width: '100%', height: '600px' }}>
  <MetahersHeroVideo />
</div>
```

> **Important:** The component fills its parent (`width: 100%`, `height: 100%`).
> Always give the parent container an explicit height, otherwise the component
> will collapse to zero height.

You can also pass a `style` prop directly to override the root container:

```tsx
<MetahersHeroVideo style={{ height: '100vh', borderRadius: '1rem' }} />
```

---

## Step 5 (optional) — Brand CSS variables

If you want the Metahers colour palette available as CSS variables for other
parts of your app (buttons, backgrounds, text), copy `MetahersHeroVideo.css`
into your `src/` folder and import it once in your root file:

```tsx
// src/main.tsx  or  src/index.tsx
import './MetahersHeroVideo.css';
```

Then use them anywhere:

```css
button {
  background-color: var(--color-accent);       /* Gold */
  color:            var(--color-text-inverse);  /* Off-white */
}

section {
  background-color: var(--color-bg-muted);     /* Soft blush */
}
```

## What the animation does

| Scene | Duration | Content |
|---|---|---|
| 0 — Brand Reveal | 5 s | "Metahers" word-mark with character stagger, gold pulse ring, ambient glow |
| 1 — Empowering | 5 s | "Empowering women / to lead & thrive" on deep navy, circle clip reveal |
| 2 — Tech | 5 s | "AI & Web3" on soft blush, rotating gold orbital rings, wipe transition |
| 3 — Leverage | 5 s | "Leverage in Business & Personal Life" on warm nude, gold diamond mark |

The loop repeats automatically, indefinitely.

---

## Customising

All colours and timing live at the top of `MetahersHeroVideo.tsx`:

```ts
// Change any palette value:
const P = {
  primary:  '#1A1A2E',   // Deep navy
  accent:   '#C9A96E',   // Gold
  // ...
};

// Change scene durations (milliseconds):
const SCENE_DURATIONS = {
  brand:      5000,
  empowering: 5000,
  tech:       5000,
  leverage:   5000,
};
```
