You are a senior full-stack engineer. Build a production-ready **installable PWA** called **MetaHers Mind Spa** as a **Next.js 14 (App Router) + TypeScript** web app. Mobile-first. Calm spa × tech-lux UI. Ship a working MVP inside this Repl.

## Stack
- Next.js 14, TypeScript, TailwindCSS
- State: Zustand
- Forms/validation: React Hook Form + Zod
- Animations: Framer Motion
- Content: MDX or inline JSON seed
- PWA: manifest.json + service worker + install prompt
- Analytics (optional): PostHog client
- No server DB for MVP. Persist minimal state in localStorage.

## Create project
1) Run:
npx create-next-app@latest metahers-mind-spa --ts --app --tailwind --eslint
cd metahers-mind-spa
npm i framer-motion zustand zod @hookform/resolvers react-hook-form next-mdx-remote posthog-js
2) Set Replit run command: in .replit set
run = "npm run dev -- --port 3000"
